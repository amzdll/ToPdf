app
├── api              - содержит веб-связанные файлы.
│   ├── dependencies - зависимости для определения маршрутов.
│   ├── errors       - определение обработчиков ошибок.
│   └── routes       - маршруты веб-приложения.
├── core             - конфигурация приложения, события запуска, логирование.
├── db               - файлы, связанные с базой данных.
│   ├── migrations   - вручную написанные миграции Alembic.
│   └── repositories - все операции CRUD (Create, Read, Update, Delete).
├── models           - Pydantic-модели для приложения.
│   ├── domain       - основные модели, используемые почти везде.
│   └── schemas      - схемы, используемые в веб-маршрутах.
├── resources        - строки, используемые в веб-ответах.
├── services         - логика, не связанная только с CRUD-операциями.
└── main.py          - создание и конфигурация приложения FastAPI.
